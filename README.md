# Elaborato sistemi distribuiti 

Repository di supporto allo sviluppo del progetto di fine corso di Sistemi Distribuiti e IOT, tenuto dal professor Russo presso l'università di Napoli Federico II nell'anno accademico 2025/26. 
In questa repository verrà sviluppato il solo codice LaTeX di supporto al progetto, la cui sorgente è reperibile a questo [link](https://github.com/h-tajato/Progetto_Distribuiti).

## Approfondimento teorico 
Nella directory approfondimenti_teorici vi sono codici sorgente e pdf già compilati di alcuni argomenti del corso che si prestavano particolarmente al tipo di trattazione discorsiva. Gli argomenti fin'ora trattati (o lo saranno presto) sono:
